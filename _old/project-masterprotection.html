Master Protection PC Privacy

Anti malware software UX|UI & branding
The project

Master Protection is a B2C anti-malware software for Windows PCs. It helps protect privacy from online threats. As a B2C product, this software is used by a variety of users and personas. The UX and UI design needs to be very understandable and friendly, as it was meant to be used by not so tech savvy users.

My role

I worked on wireframing and prototyping the UX of the website from which the users download the software, and the UI of the PC scanning software itself. I also worked on the branding and designed the logo and the graphic language.

Work process
Wireframing

Working on Figma to wireframe the first low fidelity sketches of the main screens.

Testing

Using mockup tools, I made usability testing of the screens and received users feedback.

UI Prototyping
  • Added UI of the brand's graphic language.
  • Worked on micro animations and micro interactions.
  • Worked alongside with R&D on the UI frontend development.
