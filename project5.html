The project

A website that with a series if questions measures the user's IQ, with a chance of winning prizes.

My role

I designed the UI of the website and the UX of the user activity dashboard.
The UX design proccess for the user activity dashboard was a challenging one. The users of this website are competing for the highest scores. The website's design needed to look appealing, sophisticated and very clear.

  • Wireframing & designing the website
  • Frontend development of the website
