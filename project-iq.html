IQ-Master - IQ test platform

Website design | User activity dashboard
The project

A website that hosts a test platform for measuring IQ, with a chance of winning prizes.

My role

I worked on wireframing and prototyping the UX of the website from which the users do the IQ tests and the prizes they might win. I also worked on the branding and designed the logo and the graphic language.

Work process
Wireframing

Working on Figma to wireframe the first low fidelity sketches of the main screens.

Testing

Using mockup tools, I made usability testing of the screens and received users feedback.

UI Prototyping
  • Added UI of the brand's graphic language.
  • Worked on micro animations and micro interactions.
  • Worked alongside with R&D on the UI frontend development.
IQ-Master branding and homepage

I worked on the company's branding and graphic language.
Created the logo and chose the main theme colors, fonts and style guide.
