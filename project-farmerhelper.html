Farmer's Helper

B2C Mobile Application UX & UI
The project

Farmer's Helper is a service that helps farmers to monitor their crops using cutting edge technology.

Using drones and satalite images, the software's AI composes high quality images of farms, fileds and plantations and figures out where there are pests.

My role

The main chellange of designing the mobile application was how to show the crops's major risks and the level of them, so the users (in this case the farmers) will have all the data accessed to them in a clear and a very visible way.

  • Wireframing and designing the app's user experiance
  • Designing the user interface
